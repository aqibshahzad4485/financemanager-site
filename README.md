## 📖 About
This repository contains the source code for the [Finance Manager](https://financemanager.aqibs.dev) website. Finance Manager is a highly capable, completely private, offline-first personal finance application. 

This static site serves as the central hub for the application's capabilities, documentation, open-source resources, and legal policies (Privacy Policy & Terms of Service). It features a modern, responsive "glassmorphism" aesthetic built with pure HTML, Vanilla CSS, and JavaScript.

> **Note:** The actual Android application source code is hosted separately. You can find it here: [Finance Manager App Repository](https://github.com/aqibshahzad4485/financemanager).

## ✨ Application Features

Finance Manager was built to be powerful, flexible, and completely private. The application boasts the following core capabilities:

- 🌍 **Global Multi-Currency:** Real-time exchange rates and seamless conversion across global currencies and crypto assets.
- 💱 **Smart Inter-Currency Transfers:** Easily move funds between accounts with varying base currencies without losing a cent in translation.
- 📉 **Loans & Debt Management:** Track outstanding balances, split bills, and manage complex debt structures easily.
- 🔒 **Absolute Privacy:** 100% offline architecture. Zero telemetry, zero cloud lock-in, and absolute control over your financial data.
- ♾️ **Unlimited Transactions:** No paywalls, no forced premium upgrades. Log as many expenses and incomes as you want.
- 🏦 **Wallets & Accounts:** Manage cash, checking accounts, savings, and diverse portfolios seamlessly.
- 🎯 **Advanced Budgeting:** Set strict spending limits for custom categories and monitor your budget progress in real-time.
- 📊 **Interactive Analytics:** Filter by custom durations and drill down into your spending habits with aesthetic, generative charts.
- 🎨 **Multi-Themed App:** Switch effortlessly between beautifully crafted Light and Dark modes.
- 💾 **Backup & Export:** Securely backup your database and export financial history into detailed PDF and CSV reports.

## 🛠️ Site Architecture
The website is a lightweight static site utilizing modern web design principles:
- **No Build Step:** Pure HTML/CSS/JS. No heavy frameworks like React or Vue, meaning ultra-fast load times.
- **CSS Variables:** Theming (Light/Dark mode) and brand colors are heavily reliant on modern CSS custom properties.
- **Interactive Forms:** The "Contact Us" page leverages native `mailto:` APIs for serverless communication.
- **Glassmorphism:** Employs advanced `backdrop-filter` rules for a premium frosted glass effect.
